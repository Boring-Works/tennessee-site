================================================================================
                          KNOWN ISSUES (NON-BLOCKING)
================================================================================

MINOR ISSUES:

1. Focus Return After Escape Key
   - Severity: Minor (UX Enhancement)
   - Current: Escape closes menu, doesn't restore focus to toggle
   - Expected: Focus should return to toggle button
   - Impact: Keyboard users must Tab to continue navigation
   - Recommendation: Add in next release (Priority 1)
   - Implementation: Store toggleRef, restore focus on close

2. Mobile Toggle Missing Title Attribute
   - Severity: Minor (Accessibility Enhancement)
   - Current: Button has aria-label but no title
   - Recommended: Add title="Navigation menu"
   - Impact: Minimal - aria-label already present
   - Recommendation: Add in next release (Priority 2)

DESIGN CHOICES (NOT ISSUES):

1. No Dropdown Navigation
   - Current: Flat navigation structure
   - Design: Intentional - appropriate for site structure
   - Assessment: Clean, minimalist approach
   - Future: Can be expanded when site structure grows

2. Mobile Drawer Width
   - Current: max-width: 24rem (384px)
   - Assessment: Good - leaves space to see backdrop
   - Pattern: Standard UX pattern

3. Focus Trap on Mobile
   - Current: Tab cycles within menu items
   - Assessment: Proper accessibility pattern
   - Prevents: Accidental navigation away from menu
